# Orbit Switch

A puzzle game built on [Devvit](https://developers.reddit.com/docs) (Reddit's developer platform).

Rotate interlocking rings to align their slots, then transfer coloured atoms between rings. Get every atom to its target ring to win.

## Development

```bash
npm install
npm run dev      # live-reload playtest in Devvit sandbox
npm run build    # type-check
npm run upload   # publish to Devvit
```

Requires the [Devvit CLI](https://developers.reddit.com/docs/get-started) and a Reddit account with developer access.
